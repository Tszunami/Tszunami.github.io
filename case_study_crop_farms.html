Case Study: Crops and Farms

Project is in-progress

The project plan is to outline and present current and historical trends in the acreage of crops in the U.S.

Progress:

  • Prepared gathered data for merging by formatting ~450 excel sheets through python
  • Exported and merged data into MySQL after mass converting data to csv

Due for completion:

  • Perform analysis on data looking for statistical values, trends, outliers, and anomolies
  • Identify key insights from the analysis results
  • Establish Tableau connection to MySQL server and retrieve relevant data to key insights
  • Create visuals which communicates the insights and the underlying data
